「コードは書けないけど、自分のブログを持ちたい」そんな思いから、Claude Code を使ってブログを作ってみました。その全記録をまとめます。

## Claude Code とは

Anthropic が開発した AI アシスタント「Claude」をターミナル上で操作できる CLI ツールです。日本語でやりたいことを伝えるだけで、ファイルの作成・編集・コマンド実行まで代わりにやってくれます。

## 作ったもの

- **Astro** 製の静的ブログサイト
- Markdown で記事が書ける
- ダークモード対応のデザイン
- **GitHub Pages** で無料公開

## 実際の流れ

### 1. ブログの雛形を作ってもらう

「Astro でシンプルなブログを作って」と伝えただけで、必要なファイルをすべて自動生成してくれました。自分でコードを一行も書かずに、ブログの骨格が完成しました。

### 2. GitHub Pages にデプロイする

GitHub へのプッシュや Pages の設定でいくつかつまずきましたが、エラーが出るたびに原因を調べて解決策を提案してくれました。

- GitHub アカウントが2つあり、認証でハマった
- Personal Access Token のスコープが足りなかった
- URL の生成にバグがあり、記事リンクが壊れていた

すべて Claude Code と一緒に解決できました。

### 3. 記事を追加する

記事を追加するのは簡単で、「〇〇という記事を作って」と伝えるだけ。あとは自動でファイル作成・プッシュ・デプロイまでやってくれます。

## やってみた感想

**良かった点**
- コードの知識がなくても動くものが作れた
- エラーが出ても一緒に解決してくれるので安心
- 日本語でそのまま指示できる

**難しかった点**
- GitHub の認証まわりは少し手間がかかった
- ブラウザのキャッシュで変更が反映されないことがあった

## まとめ

Claude Code があれば、プログラミングの経験がなくても自分のブログを持てることが実証できました。「何か作ってみたいけどコードは書けない」という方に、ぜひおすすめしたいツールです。

このブログ自体が、その証明です。

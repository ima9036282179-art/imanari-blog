「ドラゴン桜２」では、英語の各分野に特化した勉強法が次々と登場します。英語講師・由利杏奈（ゆりやんレトリィバァ）のユニークな指導を中心に、実践的なメソッドをまとめました。

## 単語：文脈で覚える「ストーリー記憶法」

### ドラマの教え
「単語帳だけ見て覚えようとするな。文章の中で覚えろ」

### 実践方法
1. 覚えたい単語を含む**例文を1つ作る**
2. その例文を**声に出して読む**（音読）
3. 翌日に例文を見て単語の意味が言えるか確認

| 方法 | 記憶定着率 |
|---|---|
| 単語帳だけで暗記 | 約20% |
| 例文と一緒に覚える | 約60% |
| 例文を自分で作る | 約80% |

### おすすめ単語帳
東大受験で有名な「**鉄壁**（東大英単語熟語 鉄壁）」は、例文が豊富で文脈で覚えやすい構成になっています。

## リスニング：「シャドーイング」で耳を鍛える

### ドラマの教え
由利先生が「聞き流すだけじゃダメ。声に出して追いかけろ」と指導するシーン。

### シャドーイングのやり方
1. 英語音声を流す
2. **聞こえた音をそのまま声に出してマネする**（0.5秒遅れで追いかける）
3. 最初はゆっくりの速度から始める

```
音声: "I want to go to Tokyo University..."
自分: "I want to go to..." ← 少し遅れてついていく
```

### ポイント
- 最初はスクリプト（文字）を見ながらでOK
- 慣れたら文字なしで挑戦
- 1日**15〜20分**を毎日続けることが大切

## 読解：「SVOC解析」で英文を構造で読む

### ドラマの教え
「長文は雰囲気で読むな。構造を把握してから意味を取れ」

### やり方
英文を読むとき、以下の順番で要素を確認する：

```
S（主語）→ V（動詞）→ O（目的語）→ C（補語）→ M（修飾語）
```

**例文：**
```
[S The student] [V studied] [O English] [M every day].
```

SVOCが見えると、長文でも構造が理解しやすくなります。

## 文法：「アウトプット先行」で定着させる

### ドラマの教え
「文法書を読み込んでから問題を解くのは逆だ。問題を解きながら文法を覚えろ」

### 実践方法
1. いきなり問題集を開く
2. わからなければ文法書で確認
3. 解説を読んだらすぐに類似問題を解く

「**問題 → 確認 → 問題**」のサイクルが文法習得の近道です。

---

## まとめ：英語4技能の優先順位

受験英語においてドラゴン桜２が示す優先順位：

1. **単語力**（すべての土台）
2. **文法・構造把握**（読解の核心）
3. **リスニング**（近年配点増加）
4. **英作文**（差がつきやすい）

まず単語と文法を固め、そのうえでリスニングと英作文を鍛えるのが王道です。

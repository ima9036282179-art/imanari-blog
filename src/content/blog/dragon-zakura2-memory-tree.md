「メモリーツリーって何？どうやるの？」と調べてここにたどり着いた方へ。

ドラゴン桜２で桜木先生が生徒に教えた**メモリーツリー（マインドマップ）**は、暗記が苦手な人ほど効果が出る記憶術です。この記事では**作り方の手順・コツ・科目別の活用例**を図解でわかりやすく解説します。

**この記事でわかること**
- メモリーツリーとは何か・なぜ暗記できるのか
- 実際の作り方（手順）
- 科目別の活用例（歴史・生物・英単語）

## メモリーツリーとは？

中心にテーマを書き、そこから**枝を伸ばすように関連情報を広げていく**ノート術です。

脳が情報を「ネットワーク構造」で記憶することを活かした手法で、箇条書きより記憶に残りやすいことが研究でも示されています。

---

## 基本構造（テキスト図解）

```
              ┌─── 品詞 ─── 名詞/動詞/形容詞
              │
              ├─── 語源 ─── ラテン語由来
  英単語 ────┤
  "conduct"  ├─── 意味 ─── 行動する／指揮する
              │
              └─── 例文 ─── He conducted the meeting.
```

このように、**ひとつの単語やテーマを中心に、関連情報を放射状に広げます**。

---

## 作り方：ステップ別解説

### STEP 1：テーマを中央に書く

紙の中央に、覚えたいテーマを大きく書きます。

```
        ┌─────────────┐
        │   明治維新   │  ← ここに書く
        └─────────────┘
```

---

### STEP 2：大きな枝（メインブランチ）を伸ばす

テーマから4〜6本の「大きな枝」を伸ばし、メインカテゴリを書きます。

```
             原因
              │
   人物 ───明治維新─── 出来事
              │
             結果
```

---

### STEP 3：小さな枝（サブブランチ）を追加する

大きな枝からさらに細かい情報を追加します。

```
                  ┌── 坂本龍馬
                  │
   人物 ──────┤── 西郷隆盛
                  │
                  └── 大久保利通

                  ┌── 黒船来航（1853）
                  │
   出来事 ─────┤── 大政奉還（1867）
                  │
                  └── 廃藩置県（1871）
```

---

### STEP 4：色・記号でカテゴリを区別する

手書きの場合は**色分け**することで、記憶への定着率がさらに上がります。

- 赤：人物
- 青：出来事
- 緑：年号
- 黄：結果・影響

---

## 科目別の活用法

| 科目 | 中心テーマの例 | 枝の例 |
|---|---|---|
| 日本史 | 「江戸幕府」 | 将軍一覧／制度／滅亡の経緯 |
| 英語 | 「conduct」 | 品詞／語源／例文／派生語 |
| 生物 | 「細胞分裂」 | 有糸分裂／減数分裂／各段階 |
| 数学 | 「2次関数」 | グラフ／頂点公式／応用問題 |

---

## 紙 vs デジタル

| | 紙 | デジタル（アプリ）|
|---|---|---|
| 記憶定着 | ◎（手書きの効果）| ○ |
| 修正のしやすさ | △ | ◎ |
| おすすめアプリ | ─ | XMind, MindMeister |

ドラゴン桜２の中では**手書き**を推奨しています。手を動かすことで、脳への刺激が増して記憶に残りやすくなるためです。

---

## まとめ

メモリーツリーの手順：

1. 中央にテーマを書く
2. 大きな枝（カテゴリ）を伸ばす
3. 小さな枝（詳細情報）を追加する
4. 色分けして視覚的に整理する

最初は**1つのテーマで5分**から試してみてください。慣れると、教科書1章分を1枚にまとめられるようになります。

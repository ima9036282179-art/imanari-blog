こんにちは！このブログは [Astro](https://astro.build) で作りました。

## Astro とは

Astro は、高速な静的サイトを作るためのフレームワークです。  
記事を **Markdown** で書けるので、文章に集中できます。

## できること

- Markdown / MDX で記事を書く
- コンポーネントベースの開発
- 静的ファイルとして出力するので表示が速い
- GitHub Pages に無料でデプロイできる

これからたくさん記事を書いていきたいと思います。よろしくお願いします！
